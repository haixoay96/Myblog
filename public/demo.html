Trong lập trình javascript nói chúng và nodejs nói riêng thì callback là một khái niệm rất quan trọng.
Vậy nó là gì thì hôm nay chúng ta bắt đầu tìm hiểu.
Đầu tiên chúng ta cần phải biết rằng function trong javascript cũng có thể được lưu trữ vào một biến, mà đã được lưu được vào trong biến thì đương nhiên sẽ truyền được vào trong hàm.
Chúng ta cùng xem đoạn code sau:

function printfName() {
        console.log('hello world');
};

function printf (call) {
    call();
};

printf(printfName);
// nó sẽ in ra hello world;

Chúng ta có thể hiểu rằng khi chúng ta gọi hàm printf và truyền vào tham số là printfName thì dữ liệu của hàm printfName sẽ được lưu vào biến cục bộ call và ở trong hàm printf để gọi hàm được truyền vào thì ta sử dụng cú pháp call().
Tóm lại cách chúng ta truyền hàm printfName vào trong hàm printf và sử dụng nó bằng cách gọi call() thì người ta gọi đó là hàm callback.
==>Callback tức là ta truyền 1 hàm vào trong 1 hàm khác rồi sử dụng hàm đó.
Còn đây là đoạn code ví dụ cách truyền tham số vào hàm callback!!

function printf(name) {
    console.log(name);
};
function printfName(call, name) {
    call(name);
};

printfName(call, 'Tôi là tôi!');
// nó sẽ in ra Tôi là tôi!
